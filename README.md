# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer with real-time results, score tracking, and a clean modern UI. Simple logic, smooth animations, and fully browser-based gameplay.

---

## 🚀 Features

- 🎮 Play Rock, Paper, Scissors against computer  
- 🧠 Random computer choice logic  
- 📊 Real-time score tracking (Player vs CPU)  
- 🔄 Instant result display after each round  
- 🔁 Reset button to restart the game  
- 🎨 Clean and responsive UI  
- ⚡ Lightweight and fast (no external libraries)  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

## 🎯 Usage

1. Click on Rock ✊, Paper ✋, or Scissors ✌️  
2. Computer will randomly choose its move  
3. Result will be displayed instantly  
4. Scores will update automatically  
5. Click "Reset" to restart the game

## 📌 Game Logic

- Rock beats Scissors  
- Paper beats Rock  
- Scissors beats Paper  
- Same choice = Draw
